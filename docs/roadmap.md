# Product Roadmap

## Phase 1

- Bootstrap ASP.NET Core API
- Add Swagger support
- Add first schema lesson assets
- Publish repository and baseline documentation

## Phase 2

- Add schema validation endpoint
- Return structured validation errors
- Add unit tests for valid and invalid payloads

## Phase 3

- Introduce schema catalog endpoints
- Add SQL Server persistence
- Store schema metadata and versions

## Phase 4

- Add JWT authentication and role-based authorization
- Protect schema authoring and versioning operations

## Phase 5

- Support PDF upload and text extraction
- Add document-to-JSON extraction workflows

## Phase 6

- Export validation output to JSON, CSV, and Excel
- Add downloadable validation reports

## Phase 7

- Add CI quality gates, richer tests, and release workflow
- Expand contributor documentation and issue templates
